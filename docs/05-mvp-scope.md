# MVP Scope

## In-Scope
- SMS intake
- AI intent parsing
- Basic volunteer database
- Basic shift database
- Simple matching logic
- Confirmation loop
- Staff notifications (optional)

## Out-of-Scope (for now)
- Mobile app
- Web dashboard
- Complex skill weighting
- Multi-language support
- Automated fairness balancing
- Advanced analytics

## Success Metric
A cancellation triggers:
- Automatic identification of a backup volunteer
- SMS outreach
- Confirmation
- Schedule update

All within 5 minutes, without human intervention.
